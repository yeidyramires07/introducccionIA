## Ejemplo 1: Algoritmo para encontrar al mayor de 3 números

```txt
Inicio
    leer numero1
    leer numero2
    leer numero3

    si (numero1 > numero2) y (numero1 > numero3) entonces
         mayor := numero1
    sino si (numero2 < numero3) entonces
        mayor := numero2
    sino 
        mayor := numero3
    fin si
    
    mostrar "El número mayor es ", mayor
fin
```
